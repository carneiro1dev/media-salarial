# Projeto Média Salarial

## Visão Geral

Este projeto consiste em um sistema simples desenvolvido em JavaScript para praticar lógica de programação, validação de dados e manipulação do DOM. O sistema coleta informações do usuário e realiza cálculos de média salarial com base nos valores informados.

## Objetivo do Projeto

O foco principal deste projeto é reforçar conceitos fundamentais de programação, como entrada de dados, estrutura de condições, organização de funções e retorno visual dos resultados ao usuário.

## Tecnologias Utilizadas

- HTML
- CSS
- JavaScript

## Decisões Técnicas

O projeto foi estruturado separando a parte visual da lógica de programação, facilitando a manutenção e futuras melhorias. Foram implementadas validações para evitar cálculos com campos vazios ou inválidos.

## Aprendizados

- Estruturação de funções em JavaScript
- Uso de eventos para interação com o usuário
- Manipulação de elementos HTML via DOM
- Organização de código para facilitar manutenção

## Próximos Passos

- Implementar múltiplos campos para cálculo de média mensal e anual
- Criar histórico de cálculos para o usuário
- Exibir gráficos simples de evolução salarial
- Aprimorar layout e usabilidade
